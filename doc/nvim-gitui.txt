INTRODUCTION                           *nvim-gitui-introduction*

nvim-gitui is a plugin for Neovim that provides seamless integration
with Gitui, enhancing the Git workflow within the Neovim environment.


INSTALLATION                           *nvim-gitui-installation*

To install nvim-gitui, use your preferred plugin manager.

For packer.nvim:
>
    use 'mikinovation/nvim-gitui'
<

For dein.vim:
>
    [[plugins]]
    repo = 'mikinovation/nvim-gitui'
<


CONFIGURATION                         *nvim-gitui-configuration*

To configure nvim-gitui, add the following to your `init.lua`:
>
    require('nvim-gitui')
<

More configuration details will be added soon.
