# Features

Enabling seamless operation between Gitui and Neovim

You can use Gitui on Neovim

## Prerequisites

- Neovim >= 0.8~
- Gitui is already installed

## Installation

### packer.nvim

```
use 'mikinovation/nvim-gitui'
```

### dein.vim

```dein.toml
[[plugins]]
repo = 'mikinovation/nvim-gitui'
```

## Setup

```init.lua
require('nvim-gitui')
```
